## ğŸ¯ Key Functions Explained

### 1. `findEncodings(images)`
**Purpose:** Convert face images to numerical encodings

**Process:**
```python
For each image:
    1. Load image (already in RGB from face_recognition.load_image_file)
    2. Detect face location
    3. Create 128-number encoding
    4. Store in list
Return list of all encodings
```

### 2. `markAttendance(name)`
**Purpose:** Record attendance in CSV file

**Process:**
```python
1. Open Attendance.csv
2. Read all existing entries
3. Check if name already exists
4. If NEW:
   - Get current time
   - Get current date
   - Write: name,time,date
5. Close file
```

### 3. Main Loop
**Purpose:** Continuous face detection and recognition

**Process:**
```python
While True:
    1. Read frame from webcam
    2. Resize for speed (1/4 size)
    3. Find all faces
    4. Encode all faces
    5. For each face:
       - Compare with known faces
       - Find best match
       - If match: draw box, show name, mark attendance
    6. Display frame
    7. Check for Enter key
    8. If Enter: break loop
```

## ğŸ”¢ Technical Specifications

**Face Encoding:**
- Algorithm: dlib's ResNet-based face recognition
- Encoding size: 128 dimensions
- Accuracy: ~99.38% on LFW benchmark

**Face Detection:**
- Method: HOG (Histogram of Oriented Gradients)
- Alternative: CNN (more accurate, slower)
- Detection threshold: Adjustable

**Matching:**
- Distance metric: Euclidean distance
- Threshold: 0.6 (default)
- Lower distance = better match

**Performance:**
- Frame processing: ~0.25 seconds per frame
- Webcam resolution: 640x480 (default)
- Processing resolution: 160x120 (1/4 scale)
